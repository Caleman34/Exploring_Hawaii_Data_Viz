Data & Resources

Raw Data Sets, website links, and Other info


ScrollReveal

  • As the user scrolls, various elements of the website become animated with the help of ScrollReveal.

  • ScrollReveal is a JavaScript library for easily animating elements as they enter/leave the viewport. It was designed to be robust and flexible, and is surprisingly easy it to pick up.

AniJS

  • Another JS library we use that works with scrollReveal is AniJS. AniJS provides more animation features for selected elements in a web page.

Pictures and Information

  • Throughout the website, beautiful and vibrant images were used from Go Hawaii to showcase Hawaii, along with a breif descrition of each island and Hawaii's Features.

Beach Conditions and Beach Information

  • Hawaii Beach Safety API is used for current beach information in our leaflet map showing the differebnt beach locations in Hawaii

  • The live API connection was not consistent and we ran into a CORS error often. To avoid this, we copied the .json code into a .json file to use for this project. The API link is commented out in the code.

Lodging Information

  • Hawaii geoportal was downloaded as a .geojson for hotels throughout the Hawaiian Islands.

Hawaii Precipitation and Temperature Data

  • Climate data was copied into and excel file from Weather.gov then cleaned with pandas using python in VSC then exported into an Excel file to be connected with Flask for data retreival for display on HTML.

UFO Sightings Data

  • UFO sightings data was pulled from the National UFO Reporting Center website, copied to an excel file, then converted to a js file using a CSV to json converter.

Volcano Locations (to be added soon)

  • Volcano names and locations data were copied from NOAA into an excel spreadsheet then cleaned with pandas using python in VSC then exported into an Excel file.
